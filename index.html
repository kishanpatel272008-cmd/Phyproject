Connect GitHub Storage

UniSphere will store all user data permanently in your GitHub repository.

Setup Steps:
1. Create a public or private GitHub repo (e.g. my-unisphere-db)
2. Generate a Personal Access Token with repo scope at github.com/settings/tokens
3. Paste details below — the portal will auto-create the database file.

Config is saved in your browser's localStorage. Token never leaves your device.
